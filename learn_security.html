How AES Works (Simplified)
Step 1 — SubBytes: Each byte is replaced using a fixed substitution table (S-box).
Step 2 — ShiftRows: Rows of the 4×4 block are shifted cyclically.
Step 3 — MixColumns: Columns are mixed using matrix multiplication in GF(2⁸).
Step 4 — AddRoundKey: XOR with a derived round key. Repeated 10–14 times.
